## Project Overview

This project investigates correlations between various features in a movie dataset using Python. The main objective is to determine which 
variables have the strongest relationship with movie gross earnings.

This project demonstrates practical skills in:
* Data Cleaning
* Exploratory Data Analysis (EDA)
* Correlation Analysis
* Data Visualization
* Working with Real-World Datasets



## Tools & Technologies

| Tool             | Purpose                      
| ---------------- | ---------------------------- 
| Python           | Data Analysis                
| Pandas           | Data Cleaning & Manipulation 
| NumPy            | Numerical Operations         
| Matplotlib       | Data Visualization           
| Seaborn          | Statistical Visualization    
| Jupyter Notebook | Interactive Analysis

## Dataset

The dataset contains information about movies, including:

* Budget
* Gross Revenue
* Company
* Genre
* Runtime
* Votes
* Rating
* Release Year

# Key Insights

 Budget has a strong positive correlation with gross revenue.

 Movies with higher audience votes tend to generate more revenue.

 Certain variables showed weak or almost no correlation with movie success.

Correlation heatmaps helped visualize the strongest relationships in the dataset.
